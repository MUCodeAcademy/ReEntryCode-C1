# Extra Learning

If you've completed the 3 page HTML project, here is a more advanced HTML/CSS project, as well as an introduction to JavaScript.

This project gives a good introduction to client-side scripting with JavaScript. This will require a lot of self-teaching and looking through the documentation.

## Recipe Project Overview

Create a dynamic recipe website where users can view a list of recipes, click on a recipe to see detailed instructions, and use a form to submit new recipes. This project will help you learn how to manipulate the DOM (Document Object Model) with JavaScript, handle events, and work with forms.

### Features

1. Use HTML and CSS to design a visually appealing layout for listing recipes. Each recipe in the list should display the recipe name and a short description.
2. When a user clicks on a recipe, display a detailed view of the recipe. This view should include the recipe name, ingredients, and instructions (you can just put placeholder text if you want).
3. Create a form where users can submit new recipes. The form should include fields for the recipe name, ingredients, instructions, and an image URL.
4. Add a search box to filter recipes by name. As users type in the search box, dynamically filter and display only the recipes that match the search query.
