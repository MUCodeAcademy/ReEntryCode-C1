# Practice

Do all of this with JavaScript.

1. Add a box to the screen.
2. Prompt the user for a number, then use that number as a timer to count down on the screen.
3. When the timer reaches 0, remove the box.
4. Add a button to the document. When the user clicks the button, add a different box with a green background to the screen.
5. Make an array of numbers. Display a div on the screen with each number on the screen.
    - For example, if you have the numbers 1, 2, and 3, you'll have one div with the number 1 inside of it, another div with the number 2, and another with the number 3.
