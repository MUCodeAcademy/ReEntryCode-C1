# Projects

Here are a list of projects you can try. If you have our own project you want to make, or if you want to go back and continue working on one of our previous projects, you can do that as well.

1. Digital Clock
    - Display a live digital clock on the webpage that updates every second.
    - You'll need to use the "Date" object for this.
    - An additional idea is to allow the user to set an alarm for a specific time. For example, the user sets an alarm for 7:00 am, and once the clock reaches 7:00 am, an alert displays on the page.
2. Random Quote Generator
    - Display a random quote from a pre-defined list each time the user clicks a button.
3. Blog Page
    - Create a blog page where a pre-defined list of posts can be displayed.
    - These posts can have the text content, author, date, etc.
    - Add functionality to allow the user to write and add a new post to the list.
