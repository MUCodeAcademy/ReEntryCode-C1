# Blog Page

We're going to re-create our blog page that we made on Week 3, Day 1 of JavaScript.

Here's what we need:

- Create blog posts with the content, author, likes/dislikes, date, and comments. You can add more if you want.

- Display those blog posts to the page.

- Add a button for increasing likes and dislikes for each post.

- Add an input for each post that will add a comment to that post.

- Add an input and a button that will add a new post to the page.
