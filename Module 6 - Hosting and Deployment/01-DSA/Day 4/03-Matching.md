# Matching

You're going to be building a game. In the game, the player needs to match a country to its capital by clicking on appropriate buttons.

1. A button should be displayed for each country and each capital. So, the example above would return these buttons:

Germany, Berlin, Azerbaijan and Baku

2. The buttons should be displayed in a random order.

3. Clicking a button should set its background color to blue.

4. Clicking another button should:
- remove both buttons if a correct country and capital pair has been selected.
- set the background color of both buttons to red if a wrong pair has been selected.

5. Assuming the previously selected pair was wrong, clicking another button should restore the default background color of the wrong pair and set the background color of the clicked button to blue.

6. When there are no buttons left, display a message: Congratualations!

7. Export your component as the default export.

The look of the component won't be evaluated, only its specified functionalities.
