# Character Page Steps

This page will let the user enter a character name and attributes

1. Create an input for their character name, and display that
2. Create a menu/list for different attributes (fast, strong, smart, etc.)
3. Radio buttons only let them pick one by default
<!-- <input type="radio" /> -->
4. Let them pick a (FANTASY) weapon
5. Use ItemContext for the weapons and attributes
6. Create character button at the bottom when they're done
