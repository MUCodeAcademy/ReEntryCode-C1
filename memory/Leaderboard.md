# Leaderboard

Leaderboard that stores users times to complete the game.

1. Timer to keep track of how long it takes them to complete
2. Allow the user to put in a username
3. Database for storing usernames, userId, and times to complete
4. Make a .env file to store the database information
5. Make a server.js file:
    - Stores the information
    - Gets the information for the leaderboard
6. Display the leaderboard
